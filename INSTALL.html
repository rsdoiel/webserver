Installing from source

Required software

  • deno >= 2.4.5
  • CMTools >= 0.0.40

Steps

  1. git clone https://github.com/rsdoiel/webserver
  2. Change directory into the webserver directory
  3. Make to build, test and install
git clone https://github.com/rsdoiel/webserver
cd webserver
make
make test
make install
